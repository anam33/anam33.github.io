SlideScanner Home Screen

SlideScanner

API used: AWS S3 and DynamoDB, iOS camera, OpenCV, IOBluetooth
This is the partner app for my current project in the Fletcher Lab. The core function of the app is the capture and upload images to AWS S3 and DynamoDB. The app has built in image processing in order to allow for a variety of imaging techniques. Additionally, it has control over the motors in the devices via Bluetooth.
